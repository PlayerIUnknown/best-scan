# best-scan
Best Recon Automation

## Master Recon Automation

This repository contains a simple **master recon automation** binary written in go for speed, `best-recon`, which does a massive recon on the set target.

## How to Run

1. **Clone the repository** to your machine.

3. **Run the binary**:

```bash
./best-recon <target>
```

That’s it — once executed, the script will perform recon on the target and give amazing results.
