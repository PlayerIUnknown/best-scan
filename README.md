# best-scan
Best Recon Automation

## Master Recon Automation

This repository contains a simple **master recon automation** script, `best-recon.sh`, which does a massive recon on the set target.

## How to Run

1. **Clone the repository** to your machine.
2. **Make the script executable**:

```bash
chmod +x best-recon.sh
```

3. **Run the script**:

```bash
./best-recon.sh
```

That’s it — once executed, the script will perform recon on the target and give amazing results.
